# Portfolio

This folder is your personal workspace for documenting completed projects as you work through this repository.

It is intentionally excluded from the main project structure so your work stays separate from the course exercises, troubleshoot files, and solutions.

## Suggested Structure

Organize by project, mirroring the `projects/` folder naming:

```
portfolio/
└── 01-foundational-vpc/
    ├── completed/          ← your finished scripts, Terraform, or CloudFormation files
    ├── evidence/           ← screenshots and CLI output proving deployment
    └── reflection.md       ← what you learned, what was hard, what you would do differently
```

## What to Put Here

- **Completed exercises** — your implementations of the partial files from each project's `exercise/` folder
- **Deployment evidence** — screenshots or CLI output showing resources were actually provisioned
- **Reflections** — honest notes on what clicked, what tripped you up, and what you would do differently

## Privacy Note

Avoid uploading screenshots that expose public IP addresses, account IDs, or access keys. Blur or redact sensitive values before committing.

## Getting Started

Create a folder matching the project you are working on and start adding your work as you go.
